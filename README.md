# Portfolio Website

A modern portfolio website built with React, TypeScript, Vite, and Tailwind CSS.

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Development

To get started:

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This project is deployed to GitHub Pages using GitHub Actions. The workflow automatically builds and deploys to the `gh-pages` branch on every push to `main`.
